No endorsements expressed or implied. Use at your own risk, verify information with primary sources, and follow all applicable laws and policies.

## Programming, data, and documentation

- [Python Language Reference](https://docs.python.org/3/reference/index.html)  
  Official reference for Python syntax, semantics, and language features.

- [R Reference Card (PDF)](https://cran.r-project.org/doc/contrib/Short-refcard.pdf)  
  Compact cheat sheet for common R functions and syntax.

- [R Novice Gapminder Reference](https://docs.ycrc.yale.edu/r-novice-gapminder/reference.html)  
  Intro level R examples using the Gapminder dataset.

- [Tidy Data (tidyr vignette)](https://cran.r-project.org/web/packages/tidyr/vignettes/tidy-data.html)  
  Explanation of tidy data principles and how to reshape data in R.

- [Quarto Reference](https://quarto.org/docs/reference/)  
  Documentation for Quarto formats, options, and extensions.

- [Markdown Guide Cheat Sheet](https://www.markdownguide.org/cheat-sheet/)  
  Quick reference for basic and extended Markdown syntax.

- [DSPy](https://dspy.ai/)  
  Library and documentation for building structured programs around large language models.

- [DevExpress .NET End User Docs](https://devexpress.github.io/dotnet-eud/index.html)  
  Help and reference material for DevExpress .NET controls and components.

- [GNU Project](https://www.gnu.org/)  
  Home of the GNU project with docs and downloads for many free software tools.
